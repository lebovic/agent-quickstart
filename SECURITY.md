# Security Policy

## Reporting a Vulnerability

If you discover a security vulnerability in this project, please report it by emailing **security@sproutling.dev**.

If it's serious issue, please do not disclose the issue through public GitHub issues.

Include as much of the following information as possible:

- Description of the vulnerability
- Steps to reproduce
- Potential impact
- Any suggested fixes (optional)
